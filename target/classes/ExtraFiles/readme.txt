// Descrição: Este arquivo fornece uma explicação sobre o conteúdo do diretório ExtraFiles em português e inglês.
// Description: This file provides an explanation of the contents of the ExtraFiles directory in both Portuguese and English.


— > Arquivos do diretório ExtraFiles/ - Explicação (pt_BR):

  - messages_en.yml: Este arquivo contêm as configurações de mensagens em inglês para o plugin.
  - messages_pt.yml: Este arquivo contêm as configurações de mensagens em português brasileiro para o plugin.
  - messages_en_small_caps_font.yml: Este arquivo contêm as configurações de mensagens em inglês para o plugin,
    com comentários, traduzidos e convertidos com fonte SmallCaps.
  - messages_pt_small_caps_font.yml: Este arquivo contêm as configurações de mensagens em português brasileiro para o plugin,
    com comentários, traduzidos e convertidos com fonte SmallCaps.

  * Para alterar a tradução do idioma, você deve copiar e substituir o conteúdo da configuração pelo conteúdo desejado
  no arquivo messages.yml do seu plugin e usar o comando /minifontreload para recarregar o plugin ou reiniciar seu servidor.


— > ExtraFiles folder files - Explanation (en_US):

  - messages_en.yml: This file contains the English messages settings for the plugin.
  - messages_pt.yml: This file contains the Brazilian Portuguese messages settings for the plugin.
  - messages_en_small_caps_font.yml: This file contains the English messages settings for the plugin,
    with comments, translated and converted with SmallCaps font.
  - messages_pt_small_caps_font.yml: This file contains the Brazilian Portuguese messages settings for the plugin,
    with comments, translated and converted with SmallCaps font.

  * To change the language translation, you have to copy and replace the configuration with a desired file content to
  your plugin's messages.yml file and use /minifontreload command to reload the plugin or restart your server.
