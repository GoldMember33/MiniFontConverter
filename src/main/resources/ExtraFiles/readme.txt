Extra Files:

  - config_en.yml: This file contains the English configuration settings for the plugin.
  - config_pt.yml: This file contains the Brazilian Portuguese configuration settings for the plugin.
  - config_en_small_caps_font.yml: This file contains the English configuration settings for the plugin,
    with comments and messages translated and converted with SmallCaps font.
  - config_pt_small_caps_font.yml: This file contains the Brazilian Portuguese configuration settings for the plugin,
    with comments and messages translated and converted with SmallCaps font.

  * Just copy and replace the configuration in desired file to your plugin's config.yml file and
    use /minifontreload command to reload the plugin or restart your server.
